# Swamp Preachers Game

Swamp Preachers Game is a 2D single-player and cooperative adventure based on the Swamp Preachers characters and music universe.  
Team up with a friend or face the swamp alone as you battle enemies, explore handcrafted levels, and uncover the legends surrounding the band.

## Features
- 2D pixel/art or vector-based gameplay (TBD)
- Single-player & local co-op modes
- Characters, lore, and music inspired by the Swamp Preachers band
- Progressive difficulty and unique boss encounters
- Built for fun, chaos, and teamwork
